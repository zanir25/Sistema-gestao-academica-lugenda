# Sistema de Gestão Acadêmica Lugenda

## Documentação do Projeto

Este projeto é um sistema de gestão acadêmica que visa facilitar a administração de instituições de ensino.

### Funcionalidades
- Registro de estudantes
- Gerenciamento de cursos
- Controle de notas e frequências
- Relatórios personalizados

### Instalação
Para instalar o sistema, siga os seguintes passos:
1. Clone o repositório: `git clone https://github.com/zanir25/Sistema-gestao-academica-lugenda.git`
2. Navegue até a pasta do projeto: `cd Sistema-gestao-academica-lugenda`
3. Instale as dependências necessárias.

### Uso
Para iniciar o sistema, execute o comando:
`npm start`
